⚙️ OPERAÇÕES CRUD
─────────────────────────────────────────────────────────────────────────────

CADA MÓDULO TEM:

📋 LISTAR
  - Acesse o módulo pelo menu
  - Mostra tabela com todos os registros
  - Cada linha tem ID (badge), dados e ações

🔍 BUSCAR
  - Use o campo "Buscar..." na tabela
  - Digite o nome ou descrição
  - Clique em "Buscar"
  - Resultados filtrados aparecem

➕ CRIAR
  - Clique no botão "Novo" (cor primária)
  - Preencha o formulário
  - Clique em "Salvar"
  - Volta para a listagem com novo registro

✏️ EDITAR
  - Na tabela, clique no ícone de lápis
  - Formulário abre com dados preenchidos
  - Modifique o que precisar
  - Clique em "Salvar"
  - Volta para a listagem com dados atualizados

🗑️ EXCLUIR
  - Na tabela, clique no ícone de lixeira
  - Confirmação: "Tem certeza?"
  - Clique em OK para confirmar
  - Registro é removido da listagem

🛠️ TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────

PROBLEMA: "Servidor não inicia"
SOLUÇÃO:
  - Verifique se MySQL está rodando
  - Verifique credenciais no .env
  - Verifique porta 3000 não está em uso

PROBLEMA: "Erro ao conectar ao banco"
SOLUÇÃO:
  - Confirme MySQL está rodando: net start MySQL80
  - Confirme credenciais de conexão
  - Confirme database projetos_extensao existe

PROBLEMA: "Página em branco / Erro 404"
SOLUÇÃO:
  - Verifique se fez login
  - Verifique URL está correta (case sensitive)
  - Verifique rota existe em routes/index.js

PROBLEMA: "Formulário não carrega"
SOLUÇÃO:
  - Verifique arquivo .ejs existe
  - Verifique caminho do include (../cabecalho)
  - Verifique bootstrap-icons está carregando

PROBLEMA: "Busca não funciona"
SOLUÇÃO:
  - Digite valor para buscar
  - Clique em "Buscar"
  - Se vazio, busca pode retornar tudo
